# Smart MedApp

A Spring Boot project that recommends medicines and dosages based on symptoms and age.

## Tech Stack

- Java
- Spring Boot
- Hibernate / JPA
- MySQL
- Thymeleaf
- HTML, CSS, JS

## How to Run

1. Clone the repo.
2. Set up MySQL database.
3. Update `application.properties`.
4. Run `mvn spring-boot:run`.
